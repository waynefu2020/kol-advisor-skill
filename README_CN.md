# 海外红人顾问.skill - 简介

**中文版 | [English](README.md)**

这是一个将出海营销人大威的决策框架提炼为 Claude Code 插件的 AI skill。大威拥有 8 年出海营销经验，其中 5 年专注海外红人运营，主要品类为服装与 3C 数码。

## 核心模块

**5 条红人筛选原则：**
- 评论质量优先于粉丝数量：刷量账号的评论区充斥 emoji 和无意义赞美
- TikTok 主要刷播放，Instagram 主要刷粉丝——表现不同，本质相同
- 用最近 10 条视频的播放量**中位数**而非平均数，避免爆款单视频拉高数据
- 有商业合作经历意味着内容更专业、交付更可靠
- 15 天内有更新；粉丝画像与目标人群匹配

**6 条定价与议价原则：**
- 报价参考权重：国家 > 行业 > 内容数量/时长 > 档期——美国/欧洲博主比东南亚普遍贵 2 倍以上
- 中腰部（10万–50万粉）最难谈：开始有自我认知、没有经纪公司约束、全凭感觉报价
- 遇到报价远超预算：打包价 + 佣金 + 成功案例三招组合，谈不拢就直接换下一个
- 产品置换只适合素人和 KOC，中腰部及以上不适用
- 美国博主：第一封邮件就说清楚给多少钱，别绕弯子
- 谈判时不要提博主数据哪里不好——先夸，再还价

**Brief 撰写框架：**
- 9 个模块：品牌调性 / 基础信息 / 营销目标 / 内容交付 / 必提卖点 / 文案要求 / 参考内容 / 付款方式 / 素材版权
- 素人博主：约束力强 + 给参考爆款让他们照做；KOL 及以上：给方向，不给指令
- 必提卖点不超过 3 个；修改次数不超过 2 次，且只能小改
- Brief 越细致，内容越稳定，但越难激发出好内容——早期以激发为主，后期再逐步收紧

**营销策略体系：**
- CPM 算法：历史曝光量与销售额的换算比 → 目标曝光量 → 所需预算
- 三阶段模型：内容摸索期（KOC 80%）→ 爆款打造期（KOC 60%）→ 声量放大期（KOL 60%）
- 红人营销真正的 KPI：品牌/产品搜索量 + 详情页访问量——不是 GMV，不是 ROAS

**竞品研究方法：**
- 4 步流程：选竞品 → 看产品 → 看达人矩阵 → 挖内容规律
- 工具链：Nox 聚星、EasyKOL、TikTok Creator Marketplace、Google Trends

## 工作原理

这个 skill 让 Claude 以大威的视角分析海外红人营销决策。它不给通用建议，而是把大威真实的筛选标准、议价打法和 Brief 逻辑直接用于你的具体场景。输出结构固定：先给结论，再给依据，最后给下一步动作。

## 安装与使用

```bash
npx skills add waynefu2020/kol-advisor-skill
```

安装后在 Claude Code 中输入 `/kol-advisor` 加上你的问题即可。

本项目由 [Nuwa.skill](https://github.com/anthropics/nuwa-skill) 自动生成，这是一个将思考者的认知模式提炼为可复用 AI skill 的框架。
